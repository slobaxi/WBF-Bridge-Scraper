# Bridge Tournament Simulator & Leaderboard

## Description
This is an **open-source, client-side bridge tournament simulator** that lets players practice with real-world boards from WBF tournaments and compete against official results in real time. The program makes bridge practice more engaging by allowing players to see **instant comparisons with official scores** and maintain a **dynamic leaderboard** while playing.

## Features
- Import official boards and results from WBF tournaments (JSON/LIN/Excel)  
- Play random boards locally or using a third-party bridge site  
- Enter your results (contract, declarer, tricks, etc.)  
- Compare your performance with official scores  
- Real-time leaderboard updates for one or multiple players  
- Export player results and leaderboard to CSV for sharing  
- Fully client-side, no backend required — works in any modern browser  

## Use Case
- Practice with official tournament boards  
- Simulate a full tournament experience  
- Compare your performance to world-class results  
- Friendly competitions among friends or club members
